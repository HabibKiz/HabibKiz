## 🔬 What I've Built

**[4G Congestion Detection & Prediction](https://github.com/HabibKiz/4g-congestion-detection-prediction)** — MSc Thesis · Sofrecom / Orange Group · 2025

- Processed **637,000+ rows** of live KPI export data from Orange Mali's 4G radio access network across 4 temporal granularities (Busy Hour, hourly, daily, weekly)
- Applied **HDBSCAN, K-Means, and GMM** across 11 KPI dimensions on 1M+ observation datasets — HDBSCAN achieved silhouette scores of **0.69–0.70**, surfacing actionable congestion-risk cohorts
- Benchmarked **XGBoost vs Prophet** for multi-horizon forecasting (1–7 days); XGBoost delivered **30–60% lower MAE**, establishing short-term congestion early warning as operationally feasible
- Designed a **synthetic Busy Hour dataset** using XGBoost Tweedie to extend temporal coverage from 1 to 6 months under data retention constraints

**[Deep Research Agent](https://github.com/HabibKiz/n8n-deep-research-agent)** — n8n · FireCrawl · Claude Sonnet & Opus · Notion

- Two-workflow system that replicates Anthropic's deep research feature — given a query and depth level, the agent recursively searches the web across multiple rounds and synthesises a structured report into Notion
- Orchestrator manages state accumulation across iterations; Iteration Worker handles per-round search, deduplication, extraction, and follow-up query generation
- Claude Opus for final 800–1,500+ word synthesis; Claude Sonnet for all per-URL extraction and query generation (cost-efficient at scale)
- Real-time progress visible in Notion as each research round completes

**[Iterative Content Refinement](https://github.com/HabibKiz/n8n-iterative-content-refinement)** — n8n · Claude Sonnet & Haiku

- Four-agent feedback loop (Generator → Critic → Refiner → Evaluator) that improves content until it hits a quality threshold or exhausts its attempt limit
- Achieved a **9/10 conversion score in 1 iteration** on product copy — from generic feature list to pain-led, trust-signalled copy
- Haiku for analytical tasks (Critic, Evaluator); Sonnet for creative tasks (Generator, Refiner) — quality where it matters, cost efficiency everywhere else
