## About

I build machine learning systems for telecommunications network optimisation — specifically for the 4G infrastructure that carries the majority of Sub-Saharan Africa's mobile traffic.

**MSc in Data Science and Artificial Intelligence.** Thesis completed during a 5-month internship at Sofrecom (Orange Group), where I developed an end-to-end ML pipeline for detecting and predicting 4G congestion events on live network data using unsupervised clustering (HDBSCAN, K-Means, GMM) and time-series forecasting (XGBoost, Prophet). Dataset: 637,000+ records of operational KPI exports from Orange Mali's 4G radio access network. HDBSCAN achieved silhouette scores of 0.69–0.70; XGBoost delivered 30–60% lower MAE than Prophet on 1–7 day prediction horizons.
